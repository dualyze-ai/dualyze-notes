# Aliceの2つの習慣ノートが78%同じだった話

Aliceには習慣に関するノートが2つある。

`アトミック・ハビット メモ.md` — ジェームズ・クリアーの本を読みながら書いたもの。
`習慣形成メモ.md` — 2ヶ月後、心理学の論文を調べながら別に書いたもの。

2つ目を書いたとき、1つ目の存在を忘れていた。

BobがDualyze Notesを紹介する。

---

## 問題：同じアイデアを二度書いていた

---

**Alice：**
似てそうなノートが2つあって。「アトミック・ハビット メモ」と「習慣形成メモ」。どっちを残すべきか、それとも統合すべきか分からなくて。

**Bob：**
`アトミック・ハビット メモ.md` を開いて、右クリック → 「Find similar notes」してみて。

**Alice：**
「Similar Notes」パネルが開いた。こう表示されてる：

```
類似ノート: アトミック・ハビット メモ.md

78%  マージ候補
習慣形成メモ.md
[比較]  [レポート]  [マージ草稿を作成]
```

**Bob：**
78%は強い重複。マージ候補の範囲。

**Alice：**
2ヶ月ずらして書いたのに、こんなに重なってるとは思わなかった。

**Bob：**
内容は違うかもしれない。でも語彙・見出し・リンクの78%が一致してる。何が重なっていて、何が違うのかを見てみよう。

---

## ステップ1：並べて比較する

---

**Alice：**
「比較」を押したら、2つのノートが並んで表示された。

スコアの内訳：

```
総合: 78%

見出し: 85%   キーワード: 82%   タグ: 70%
リンク: 60%   タイトル: 40%
```

見出しが85%一致してる。だから入れ替えても気づかないわけだ。

**Bob：**
キーワードチップを見てみて。

**Alice：**
**共通：** `習慣`, `きっかけ`, `ルーティン`, `報酬`, `アイデンティティ`, `行動`, `摩擦`, `環境設計`, `習慣ループ`, `キーストーン習慣`

**アトミック・ハビット メモのみ：** `ジェームズ・クリアー`, `4つの法則`, `気づきやすくする`, `魅力的にする`, `簡単にする`, `満足感を与える`, `2分間ルール`, `習慣の積み重ね`

**習慣形成メモのみ：** `BJ フォッグ`, `タイニー・ハビット`, `モチベーションの波`, `アンカー習慣`, `お祝い`, `神経可塑性`, `ドーパミン`, `大脳基底核`

**Bob：**
そこが答え。どちらも習慣のノートだけど、引用している著者が違う。ジェームズ・クリアーとBJフォッグ。フレームワークも違う。

**Alice：**
「習慣形成メモ」がBJフォッグの理論に基づいてたとは気づかなかった。心理学の記事を読んで書いたやつで、アトミック・ハビットとは別物だと思ってた。

**Bob：**
これが最もよくあるパターン。同じトピック、異なる出典、異なる時期に書いた。重複に見えるけど、実は同じ主題に対する2つの視点。

> **💡 スコアの内訳について**
> Dualyze Notesは5つの次元でスコアを算出します。見出しの一致率が高い＝同じ構造を持つ。キーワードの一致率が高い＝同じ概念を扱っている。「それぞれのノートのみ」のキーワードが、削除したら失われるものを教えてくれます。

---

## ステップ2：マージ草稿を作る

---

**Alice：**
「マージ草稿を作成」を押した。

`Dualyze Merge Drafts/` に新しいファイルができてた：`アトミック・ハビット メモ + 習慣形成メモ — Merge Draft.md`

中身はこんな感じ：

```markdown
# アトミック・ハビット メモ + 習慣形成メモ — Merge Draft

_Dualyze Notesで生成 — 使用前に内容を確認してください_

## 共通内容

### 習慣ループ
きっかけ → ルーティン → 報酬...

### アイデンティティベースの変化
習慣は「なりたい自分」への投票...

### 環境設計
摩擦を減らす、良い習慣を明確にする...

## アトミック・ハビット メモのみ

### 行動変容の4つの法則
気づきやすく / 魅力的に / 簡単に / 満足感を与える...

### 2分間ルール
どんな習慣も2分でできる形に縮小する...

### 習慣の積み重ね
既存の習慣に新しい習慣を接続する...

## 習慣形成メモのみ

### BJフォッグのタイニー・ハビット
アンカー習慣、モチベーションの波、お祝いの重要性...

### 神経科学的な根拠
ドーパミン、大脳基底核、神経可塑性...

---
参照元: [[アトミック・ハビット メモ]], [[習慣形成メモ]]
```

**Alice：**
元のノートは変わってない？

**Bob：**
常に。マージ草稿は編集のための「たたき台」。何を残すか、削るか、書き直すかはAliceが決める。

**Alice：**
これがまさに必要だったもの。クリアーの4つの法則とフォッグのタイニー・ハビットが並んで見える。互いに補完し合ってる。

**Bob：**
重複に見えたから削除しようとしてた。でも実は、同じトピックの異なる角度を扱ってたんだ。

**Alice：**
2つを手で読み比べてたら、BJフォッグとの接続は気づかなかったと思う。長いノートの途中に埋まってたし。

**Bob：**
「それぞれのノートのみ」のセクションが、削除したら失われるものを表面に出してくれる。それがポイント。

> **💡 マージ草稿とは**
> マージ草稿は `Dualyze Merge Drafts/` に生成されるたたき台ドキュメントです。共通内容とそれぞれのノートに固有の内容を分けて整理します。元のノートは一切変更されません。草稿はあくまで出発点です。

## 全体の流れ

```text
アトミック・ハビット メモ.md ＋ 習慣形成メモ.md
        ↓
  類似ノートを検索
  （類似度: 78% — マージ候補）
        ↓
  並べて比較
  （共通キーワード ＋ それぞれのノート固有のキーワード）
        ↓
  マージ草稿を作成
  （Dualyze Merge Drafts/ に生成 — 元ノートは変更なし）
        ↓
  編集・判断
  （両方残す？統合？クロスリンクを追加？）
```

## おわりに

---

**Alice：**
「習慣形成メモ」は重複だと思って削除しようとしてた。

**Bob：**
何が変わった？

「アトミック・ハビット メモ」が4つの法則のフレームワークを扱って、「習慣形成メモ」が神経科学とBJフォッグのメソッドを扱ってるって分かった。2つ合わせると、どちらか1つより完全になる。

**Bob：**
マージ草稿があることで、推測ではなく確認のうえで判断できる。

**Alice：**
生産性に関するノートのフォルダも同じ問題が起きてそう。次はそこで試してみる。
