## How to use

1. Open a note — start with **Spaghetti Carbonara** or **Tokyo Travel Plan**.
2. Right-click the note tab or the file in the explorer and choose **Find similar notes**.
3. The Similar Notes panel opens on the right. Try the other actions from there.
4. Adjust the threshold slider to see more or fewer results.

See the full walkthrough in the [plugin README](https://github.com/dualyze-ai/dualyze-notes#tutorial).
